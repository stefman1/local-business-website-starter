# CLAUDE.md

## Project
This is a starter website template for local service businesses.

## Goal
Build clean, professional websites that help businesses get more calls, leads, form submissions, and bookings.

## Design Style
- Clean
- Classy
- Mobile-first
- Professional
- Premium but not flashy
- Easy to read
- Strong spacing
- Clear call-to-action buttons

## Avoid
- Cheesy AI copy
- Too many animations
- Overcomplicated code
- Huge walls of text
- Cheap-looking gradients
- Random features that do not help the business get leads

## Website Sections
Most sites should include:
- Hero section
- Services section
- About section
- Trust/proof section
- FAQ section
- Contact section
- Clear call-to-action buttons

## Code Rules
- Keep the code simple and organized
- Explain major changes clearly
- Do not delete important files without asking
- Make the site responsive for phones
- Make the smallest clean fix before rewriting everything

## My Skill Level
I am learning website building and AI tools. Explain steps clearly, but keep it practical.
