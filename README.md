# Local Business Website Starter

Astro starter project for local service business websites.

## Goal

Provide a reusable base for fast, clean local business websites focused on calls, bookings, leads, and trust.

## Current Stack

- Astro
- Static site workflow

## Useful For

- Service business landing pages
- Practice websites
- Layout experiments
- Starter components

## Website Priorities

- Mobile-first structure
- Clear CTA
- Strong hero
- Services section
- Reviews/proof
- FAQ
- Footer with contact info

## Status

Keep as old starter repo. Fold the best ideas into local-business-website-template later.
